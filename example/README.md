# stt_flutter_example

Demo app for `stt_flutter` — fully local on-device speech-to-text.

## Usage

1. Select a model from the list (models download on first use)
2. Tap the mic button to start recording (grant microphone permission)
3. Speak, then tap stop
4. Transcription appears below the button, with detected language badge
5. Use "Test with sample audio" to verify the engine works

## Models

All models are downloaded at runtime from HuggingFace / GitHub on first use.
No bundled (embedded) model files are included in the example app.
